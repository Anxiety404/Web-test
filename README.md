# DeSaak Home-page
[DeSaak](https://desaak.com/) homepage but with my own little spin.

## Installation
```bash
cd path/to/Github/Web-test/front-end
php -S localhost:8080
Localhost URL: http://localhost:8080/Github/Web-test/front-end/index.php
```

## Features
- Secure sessions & CSRF protection – All user interactions are safe and session-based.

- Real-time chat with usernames – Users can set a name and chat instantly, with messages saved in the database.

- Censorship & moderation – Banned words are automatically censored in messages on refresh.

- Persistent chat history – All messages include username, page, and timestamp in MySQL.

- Responsive, animated frontend – Smooth animations, flexible layout, and visually appealing design.

- Interactive homepage sections – “Slimmer werken”, “Sneller resultaten”, and “Samen groeien” with links for more info.

- GitHub integration – Footer with direct link to the project repository.
